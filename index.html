Featured Project

Java Chat Bot On Freenode

This chat bot is built in java and connects to a freenode channel. The JavaChatBot will respond in chat any time a user sends a message that matches a known request. The 4 known request types are: time, zip, humans and weather. The JavaChatBot responds to a time request by printing a java.util.date object to string. The remaining 3 requests require API calls to other websites. JavaChatBot formats the API return values from JSON to string and prints into the channel.
