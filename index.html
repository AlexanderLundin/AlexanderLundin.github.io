Featured Project

Cloud hosted Spring Boot RESTFul Api with Front end and Back end

Check out this Rest API. The frame below is a fully interactive embedded frame that holds another webpage inside. Try clicking buttons embedded in the frame below.

This web app is built in java. It has two parts. The part embedded in the frame above is the front-end. It is a React web page. The front-end is hosted on AWS S3, for static hosting of web pages. The second part is the back-end, also cloud hosted, but at a different endpoint. The back-end is hosted on AWS Elastic Beanstalk. It is a Microservice API written in Java with Spring boot. The back-end is CORS enabled so the front-end can access the data, even though they are hosted at different domains.
